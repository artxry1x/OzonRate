№1 OzonRate

Легкое расширение для браузера, конвертирующее цены на Ozon и Ozon Банке в валюту по выбору (EUR, USD, RUB).

 !! Возможности !!
• Конвертация цен в реальном времени.
• Поддержка Ozon Банка.
• Валюта конвентируется в реальном времени.

Установка
1. Скачайте этот репозиторий (Code -> Download ZIP).
2. Откройте `chrome://extensions/`.
3. Включите «Режим разработчика».
4. Нажмите «Загрузить распакованное расширение» и выберите папку.
