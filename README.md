№1 OzonRate

Легкое расширение для браузера, конвертирующее цены на Ozon и Ozon Банке в валюту по выбору (EUR, USD, RUB).

 !! Возможности !!
 1. • Конвертация цен в реальном времени.
 2. • Поддержка Ozon Банка.
 3. • Валюта конвентируется в реальном времени.

Установка
1. Скачайте этот репозиторий (Code -> Download ZIP).
2. Распакуйте архив в удобную Вам папку.
3. Откройте `chrome://extensions/`.
4. Включите «Режим разработчика».
5. Нажмите «Загрузить распакованное расширение» и выберите папку.
